# Valentine's Day Promotional Images

## Required Images

Please place your Valentine's Day promotional images in this directory with the following names:

### 1. radhvi-valentine-white.png
- **Usage**: Hero banner, light backgrounds
- **Recommended size**: 800x200px (or similar aspect ratio)
- **Background**: White or transparent
- **Format**: PNG with transparency
- **Max file size**: 200KB (compressed)

### 2. radhvi-valentine-black.png
- **Usage**: Promotional section, dark backgrounds
- **Recommended size**: 800x200px (or similar aspect ratio)
- **Background**: Black or transparent
- **Format**: PNG with transparency
- **Max file size**: 200KB (compressed)

### 3. radhvi-valentine-icon.png
- **Usage**: Announcement bar, category card icon
- **Recommended size**: 200x200px (square)
- **Background**: Transparent
- **Format**: PNG with transparency
- **Max file size**: 50KB (compressed)

## Image Optimization Tips

1. **Compress your images** using tools like:
   - TinyPNG (https://tinypng.com/)
   - ImageOptim (Mac)
   - Squoosh (https://squoosh.app/)

2. **Ensure transparency** is preserved for PNG files

3. **Test on different backgrounds** to ensure visibility

4. **Check file sizes** - smaller is better for performance
